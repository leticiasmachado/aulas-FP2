Usando a biblioteca PrimeFaces

Entre no link no moodle e faça o download do arquivo “primefaces-12.0.0-jakarta.jar”. Crie uma pasta no projeto chamada “libraries” e adicione o arquivo na pasta, depois vincule-o ao seu projeto no NetBeans.

Acesse o site www.primefaces.org → menu → demo. Copie os componentes que forem do seu interesse. Copie o componente input text e insira-o nas tags que já criamos anteriormente



Para que funcione corretamente, é necessário colocar o link xmlns:p="http://primefaces.org/ui" na tag html. Para que o botão de adicionar atualize a tabela, colocaremos o código abaixo e definiremos o id “tabela” na tag "p:dataTable".
